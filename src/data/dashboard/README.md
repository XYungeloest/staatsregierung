# Dashboard-Daten

Die Dateien in diesem Verzeichnis sind kuratierte Sekundärquellen für interaktive Start-, Themen- und Haushaltsmodule.

Sie dienen insbesondere:

- dem 15-Punkte-Dashboard
- der Projekt-Timeline
- dem Gesetzgebungs-Tracker
- dem Haushalt-Explorer

Wichtig:

- Diese Daten sind nicht die primäre redaktionelle Wahrheit des Portals.
- Primäre Stammdaten liegen weiterhin unter `content/`.
- Verweise auf Themen, Ressorts, Presse oder Normen sollten nach Möglichkeit über bestehende Slugs und Routen hergestellt werden.
- Wenn Inhalte an mehreren Stellen gepflegt werden müssen, ist mittelfristig eine stärkere Ableitung aus Collections vorzuziehen.
